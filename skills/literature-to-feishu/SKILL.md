# 文献整理到飞书 Skill

## 何时使用
- 整理某个主题的文献
- 创建文献综述文档
- 将 Zotero 论文导出到飞书
- 团队分享研究进展

## 工作流程

### 步骤 1: 从 Zotero 获取文献
```
使用 zotero MCP:
- zotero_search: 搜索特定主题
- zotero_get_collection_items: 获取文献夹内容
- zotero_get_recent: 获取最近添加
```

### 步骤 2: 信息提取
对每篇论文提取：
- 基本信息（标题、作者、年份）
- 核心贡献（一句话）
- 方法概述（2-3句）
- 关键结果

### 步骤 3: 分类整理
按以下维度组织：
- 时间线：发展历程
- 方法分类：不同技术路线
- 问题分类：解决的具体问题
- 重要性：影响力排序

### 步骤 4: 输出到飞书
```
使用 lark-mcp:
- docx.document.create: 创建文档
- docx.document.block.create: 添加内容块
- bitable.app: 创建表格视图
```

## 飞书文档模板

### 文献综述文档结构
```markdown
# [主题] 文献综述

## 概述
- 研究背景
- 核心问题
- 文献数量和范围

## 发展脉络
### 早期工作 (年份)
### 关键突破 (年份)
### 最新进展 (年份)

## 方法分类
### 类别1
| 论文 | 方法 | 优点 | 局限 |
|------|------|------|------|

### 类别2
...

## 代表性工作详解
### [论文1标题]
...

## 开放问题与未来方向

## 参考文献
```

### 多维表格结构
| 字段 | 类型 | 说明 |
|------|------|------|
| 标题 | 文本 | 论文标题 |
| 作者 | 文本 | 第一作者 |
| 年份 | 数字 | 发表年份 |
| 会议/期刊 | 单选 | 发表渠道 |
| 主题 | 多选 | 研究主题标签 |
| 方法类型 | 单选 | 方法分类 |
| 一句话总结 | 文本 | 核心贡献 |
| 阅读状态 | 单选 | 未读/略读/精读 |
| 重要性 | 评分 | 1-5星 |
| 笔记链接 | 链接 | 详细笔记 |

## 批量处理流程

1. **获取文献列表**
   - 从 Zotero collection 获取
   - 或按关键词搜索

2. **批量提取信息**
   - 遍历每篇论文
   - 提取元数据和摘要
   - 生成一句话总结

3. **创建飞书表格**
   - 使用 bitable API 创建
   - 填充数据

4. **生成综述文档**
   - 创建文档结构
   - 填充各部分内容

## 注意事项

- 单次处理不超过 20 篇（避免上下文过长）
- 优先使用表格形式（便于后续筛选）
- 保留 Zotero 原始链接（便于回溯）
- 定期同步更新
