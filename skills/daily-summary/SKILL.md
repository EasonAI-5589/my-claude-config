# 每日总结 Skill

> 生成每日工作总结，保存到本地并导出到飞书

## 触发场景
- 用户说 "总结今天" / "今天做了啥" / "每日总结"
- 用户说 "/daily-summary"

## 工作流程

### 1. 收集信息
```bash
# 读取今日会话历史
~/.claude/projects/*/$(date +%Y%m%d)*.jsonl

# 读取待办清单
~/.claude/todos.md

# 读取草稿目录
~/.claude/drafts/
```

### 2. 分类整理
按以下板块组织内容：
- **科研** - 论文阅读、复现、写作
- **小红书** - 内容创作、发布
- **Claude 配置** - MCP、Skills、Hooks
- **工具调研** - 新工具、API
- **面试准备** - 如有相关任务
- **其他**

### 3. 生成总结模板

```markdown
# 每日总结 - YYYY-MM-DD

## 📊 会话统计
| 项目目录 | 会话数 |
|---------|--------|
| xxx | n |

---

## 🔬 板块一：科研
### 完成
- [x] xxx

### 进行中
- [ ] xxx

---

## 📋 待办进度
### 新增
- xxx

### 完成
- xxx

### 仍在进行
- xxx

---

## ✅ 今日成果
-

## 🔄 明日计划
-

---
*Generated at HH:MM*
```

### 4. 保存和导出
1. 保存到 `~/.claude/daily-summaries/YYYY-MM-DD.md`
2. 使用 lark-mcp 导出到飞书（可选）
3. 发送邮件通知（可选）

## 输出要求
- 简洁明了，重点突出
- 待办状态清晰
- 包含明日计划
