# Fetch MCP

获取网页内容，将 HTML 转换为 Markdown。

## 安装

```bash
# 需要 Python + uv
claude mcp add fetch -- uvx mcp-server-fetch
```

## 用途

- 获取指定 URL 的内容
- 将网页转换为 Markdown 格式
- 分析网页内容

## 示例

```
"获取 https://example.com 的内容"
→ Fetch 会返回网页的 Markdown 内容
```
