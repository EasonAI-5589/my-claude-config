# Daily Summary Generator

请帮我生成今天的工作总结。

## 任务
1. 回顾我们今天在这个会话中讨论和完成的所有内容
2. 按以下格式生成总结：

```markdown
# 每日总结 - [日期]

## 今日完成
- [列出完成的主要任务]

## 进行中
- [列出还在进行的任务]

## 关键发现/学习
- [今天学到的重要内容]

## 明日计划
- [基于今天的进展，建议明天做什么]
```

3. 将总结保存到 `~/.claude/daily-summaries/YYYY-MM-DD.md`

## 注意
- 如果存在昨天的总结文件，先读取它来了解上下文
- 保持简洁，每个类别最多 5 条
